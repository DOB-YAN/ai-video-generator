# 🚀 Reapersaint AI Shorts Factory

## Quick Launch Guide
1. Create a free account on **Vercel.com** and **Supabase.com**.
2. Connect your GitHub repository to Vercel.
3. Paste your free API keys into the Vercel Settings:
   - `GROQ_API_KEY`: Get from console.groq.com
   - `HF_TOKEN`: Get from huggingface.co/settings/tokens
   - `DATABASE_URL`: Get from Supabase project settings.

## Features
- Unlimited free generations.
- High-quality 1-minute video shorts.
- No-cost background music and voiceovers.
